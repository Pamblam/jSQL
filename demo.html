Live Demo - Calendar

Use the form to add events to the calendar. Click an event to view or cancel it. Click on "Reset Calendar" to delete all events. If you navigate away from this page your calendar will be the same when you return. In addition to jSQL, this demo uses calendarJS, jsDateFormatter, and DateTimePicker. There are more examples in the Github project you can check out.
